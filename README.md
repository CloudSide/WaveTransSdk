WaveTransSdk (声波传输SDK)
============

c /
- freq_util (频率管理)
- kiss_fft (FFT)
- rscode (Reed–Solomon error correction)

javascript /
- chirp.js (声音发送端的js的实现)
- example.html (调用实例)

php /
- rscode (Reed–Solomon error correction PHP版本的实现)
- wave (音频流生成器, PHP版本的实现)

objc /
- PCMRender (音频流生成器, Objective-C版本的实现)
